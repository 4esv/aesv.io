# aesv.io

My corner of the internet. Built to share info about me and the things I care about.

Monospace. Two columns on desktop, one on mobile. Plain text for terminal clients.
Live data from Spotify, Strava, and Chess.com.

Built with [Fastify](https://fastify.dev) and [Nunjucks](https://mozilla.github.io/nunjucks/).

Want a blank version? See [`clean-template`](https://github.com/4esv/aesv.io/tree/clean-template).

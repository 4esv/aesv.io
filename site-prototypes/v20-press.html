§ work

Every project I've built here started the same way: someone had a manual process that was eating their time, and no off-the-shelf tool could fit the specific shape of their problem. So I built custom ones.

Interfolio Dossier Builder
2024 · React · Node.js · BullMQ · Python · PyPDF2 · MSSQL

Faculty tenure review used to be a multi-day ordeal: coordinators collecting documents by hand, sorting them, merging PDFs, emailing everything around. I built a React frontend where staff submit jobs, an Express API queues them through BullMQ and Redis, and a Python worker handles all the document work — authenticated API calls, regex classification, PDF merging, nested bookmark generation.

Multi-day manual process → 5 minutes. 23,000 lines, 6 languages, 127 files.
Device Asset Sync
2023 · PowerShell · Python · Jamf · Snipe-IT · OAuth · MDM

1,300 assets across two management systems that had never spoken. I wrote a bi-directional PowerShell sync with proper OOP structure — exponential backoff, OAuth token refresh, request throttling — and deployed a Windows logon agent via Group Policy. Now handling 40,000+ devices.

Inventory accuracy: unreliable → 95%+. Weekly manual reconciliation: eliminated.
App Security Manager
2023 · Node.js · MSSQL · LDAP · SAML · RBAC

A ColdFusion system that had been running for over a decade needed to die. I built its replacement: three-tier RBAC dashboard with live LDAP search, drag-and-drop user assignment, transactional SQL, and Shibboleth/SAML authentication. Permission changes that used to require a sysadmin email chain now take under a minute.

Two colleges served. Access changes under 1 minute. 3,400 lines.
NAO Bot
2022 · Python · OpenAI · Whisper · NAOqi · Aldebaran

A research team needed a humanoid conversation demo for an NSF grant. I wired NAO's microphone to Whisper, Whisper to GPT, GPT to NAO's speech engine with synchronized gestures. Full speech-to-speech AI loop on humanoid hardware.

Proof-of-concept delivered. NSF grant supported.
